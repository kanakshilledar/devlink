Package handler

import "devlink/handler"

Overview ▹

Overview ▾

Package handler provides functionalities for managing events in a MongoDB database. This package includes methods for user authentication, event creation, retrieval, updating, and deletion.

Package handler manages user-related operations such as creating, logging in, and updating users. It interacts with the MongoDB database and handles JWT token generation for authentication.
